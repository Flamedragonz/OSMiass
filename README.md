# OSMiass

Эта программа предназначена для рисования и редактирования пространственных объектов на карте.

## Новые возможности

- **Динамические точки пересечения**: при пересечении рёбер отображаются временные точки,
  которые можно подтвердить, чтобы разбить рёбра и полигоны. Отображение и создание
  точек включается/выключается флажком «Динамические точки».
- **Динамическое разделение наложений**: если полигоны накладываются, большая фигура
  автоматически теряет пересекаемую часть (суммарная площадь не складывается).
  Для работы нужна галочка «Динамическое деление», которая активна только при
  показе динамических точек.
- **Слияние полигонов**: при удалении ребра, разделявшего два полигона, они снова
  соединяются, а не исчезают.
- **Фикс пустой карты**: нижняя панель управления теперь имеет минимальную высоту, чтобы
  элементы не «схлопывались» при запуске.

## Установка

Зависимости перечислены в `requirements.txt`. Новая функциональность требует
[shapely](https://pypi.org/project/Shapely/), который устанавливается вместе с остальными
пакетами.
